# ByteCargo 🚚

ByteCargo is a full-stack logistics bidding platform where **Merchants** post shipment requests and **Transporters** submit competitive offers with pricing and ETA.

It streamlines shipment allocation through a marketplace model with role-based dashboards, authentication, and responsive modern UI.

---

## 🌐 Live Demo

🔗 Frontend: https://bytecargo-one.vercel.app
🔗 Backend API: https://bytecargo-api.onrender.com

---

## ✨ Features

### 🔐 Authentication

* Secure Signup / Login
* Role-based access control
* Protected routes

### 🧑‍💼 Merchant Features

* Create shipment requests
* View all created bids
* Review transporter offers
* Accept best transporter
* Manage shipment flow

### 🚛 Transporter Features

* Browse shipment opportunities
* Submit offers with price & ETA
* Track submitted offers
* Dedicated dashboard

### 🎨 UI / UX

* Responsive dashboard UI
* Collapsible sidebar navigation
* Interactive landing page
* Clean modern design
* Mobile-friendly layout
