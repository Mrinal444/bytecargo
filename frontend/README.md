# ByteCargo Frontend

Simple React + Vite frontend for the ByteCargo backend.

Quick start:

1. Change to the `frontend` folder:

```bash
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

Environment:
- The frontend expects the backend at `http://localhost:3000` by default. To override, set `VITE_API_BASE` in your environment.

Primary pages:
- Landing, Login, Signup, Merchant dashboard, Transporter dashboard, Bid details, My bids, My submissions, Profile.
